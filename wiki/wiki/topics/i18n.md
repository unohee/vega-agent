# UI Internationalization (i18n)

v0.1.6 added a KO/EN toggle. A 4-phase roadmap lives in `docs/I18N_ROADMAP.md`.

## Current Implementation (Phase 0)

- `data-i18n`, `data-i18n-title`, `data-i18n-placeholder` attribute pattern
- `VEGA_STRINGS` object (ko/en translation pairs)
- `applyLang()` function for batch DOM replacement
- Selected language persisted via `localStorage['vega_lang']`
- `chat.html`: 25 static elements marked
- `dashboard.html`: 10 card titles marked

## Incomplete Areas (Planned for Phase 1)

- Dynamically generated text in the MCP modal and LLM modal
- Toast messages
- Agent response language linkage (Phase 4)

## Adding a New Language

Add a language code + translation pair to `VEGA_STRINGS`.
A dropdown switcher is planned for Phase 3 (currently a KO↔EN toggle button).
