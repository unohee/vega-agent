# UI 다국어 지원 (i18n)

v0.1.6에서 KO/EN 토글 추가. `docs/I18N_ROADMAP.md`에 4단계 로드맵.

## 현재 구현 (Phase 0)

- `data-i18n`, `data-i18n-title`, `data-i18n-placeholder` 속성 패턴
- `VEGA_STRINGS` 객체 (ko/en 번역 쌍)
- `applyLang()` 함수로 DOM 일괄 교체
- `localStorage['vega_lang']`으로 선택 언어 지속화
- `chat.html`: 정적 요소 25개 마킹
- `dashboard.html`: 카드 타이틀 10개 마킹

## 미완성 영역 (Phase 1 예정)

- MCP 모달, LLM 모달 동적 생성 텍스트
- 토스트 메시지
- 에이전트 응답 언어 연동 (Phase 4)

## 새 언어 추가

`VEGA_STRINGS`에 언어 코드 + 번역 쌍 추가.
Phase 3에서 드롭다운 전환 예정 (현재 KO↔EN 토글 버튼).
