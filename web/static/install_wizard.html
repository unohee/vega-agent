VEGA 설치

OpenRouter 연결

VEGA는 OpenRouter를 통해 LLM에 연결됩니다. API 키를 입력하면 연결된 LLM이 나머지 설정을 직접 안내합니다.
키가 없으면 openrouter.ai/keys에서 발급하세요.

설정 어시스턴트

연결된 LLM이 초기 설정을 진행합니다.

Google 연동 (선택)

Gmail·Calendar·Drive 도구를 쓰려면 Google Cloud OAuth를 연결하세요. 건너뛰어도 나중에 설정할 수 있습니다.
Google Cloud Console → API 및 서비스 → 사용자 인증 정보에서 "데스크톱 앱" OAuth 클라이언트를 만드세요.
