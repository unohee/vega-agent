VEGA 설치

LLM 프로바이더 연결

VEGA가 사용할 추론 백엔드를 고르세요. 연결되면 그 LLM이 나머지 설정을 직접 안내합니다.
브라우저가 열리면 로그인하세요. 키 입력은 필요 없습니다.

설정 어시스턴트

연결된 LLM이 초기 설정을 진행합니다.

Google 연동 (선택)

Gmail·Calendar·Drive 도구를 쓰려면 Google Cloud OAuth를 연결하세요. 건너뛰어도 나중에 설정할 수 있습니다.
Google Cloud Console → 사용자 인증 정보 → "데스크톱 앱" OAuth 클라이언트.

Slack 연동 (선택)

Slack 채널·DM을 읽고 검색하려면 워크스페이스에 연결하세요. 건너뛰어도 나중에 설정할 수 있습니다.
"Slack 연결"을 누르면 새 탭에서 워크스페이스 동의 화면이 열립니다. 동의 후 이 창으로 돌아오면 자동으로 완료 처리됩니다.
